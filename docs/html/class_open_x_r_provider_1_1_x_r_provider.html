◆ OpenXRInit()

void OpenXRProvider::XRProvider::OpenXRInit ( bool  bEnableDepthTextureSupport = false)
private

Initialize OpenXR - establish a connection to an active runtime. (1) Set application general info (e.g. App name, version, engine name, etc) (2) Check runtime's supported extensions and tag ones selected by the app to enable (e.g. Graphics type, Visibility Mask, Handtracking, etc), (3) Create an instance (object that allows communication to the runtime) and (4) Get xr system (representing a collection of physical xr devices) from active OpenXR runtime
