Sandbox app for the OpenXR Provider library
Demo app for the OpenXRProvider library - A shared library that provides classes that when instantiated, exposes to an app simpler access to any currently running OpenXR runtime (e.g SteamVR, OculusVR, etc) by abstracting away most of the necessary OpenXR calls and their strict call sequence
